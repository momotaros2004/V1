ผมต้องการนำความรู้ไปพัฒนาต่อและทำงานในบริษัทดีๆที่สนใจไว้้ ไม่ว่าจะเป็นการสร้างหรือออกแบบ Appication การเขียน Code ออกแบบโดยใช้ทั้ง Front-end และ Back-end ไปพัฒนา

ความคาดหวังต่อวิชานี้

ผมคาดหวังว่าต้องมีความรู้ที่สามารถนำไปใช้ทำงานได้จริงทั้งตอนฝึกงานละตอนที่ลงมือทำงานกับบริษัทได้จริงๆ

จุดอ่อนด้าน Programming และแนวทางการพัฒนา

จุดอ่อนของผมคือการที่ผมนั้นไม่สามารถรับข้อมูลความรู้ต่างๆเข้ามาพร้อมกันได้ จึงทำให้ตัวเองนั้นพัฒนาทุกด้านได้ช้ามากกว่าการเรียนรู้ด้านใดด้านหนึ่ง แนวทางการพัฒนาคือพยายามมากขึ้นเพื่อที่จะเรียนรู้และแก้ไขปัญหาต่างๆที่ไม่เข้าใจ
